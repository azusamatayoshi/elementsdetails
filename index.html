a ハイパーリンクを設定する
abbr 略語であることを示す
address 連絡先を示す
area イメージマップを設定する際に使用
article それだけで完結している内容の記事を示す
aside 文書のメインコンテンツとは関連性の低いセクションであることを示す
audio 文書の中に音声データを埋め込む
b 読者の注目を集める意味で使用し、特に重要性を示すものではない
base 文書中で使用される全ての相対URLの基準となるURLを指定する
bdi 要素内容の文字列が他とは別の文字表記の方向であることを示す
bdo 現在の文字表記の方向を上書きし、要素内容の文字列の表記方向を明示的に指定する
blockquote その段落が引用されたものであることを示す
body ブラウザに表示される文書内容を示す
br 改行を表す
button クリックできるボタンを表す
canvas 図形やアニメーションを表示させるために使用
caption 表のタイトルを表す
cite 文書の中で引用または言及した作品のタイトルを表す
code 要素内容の文字列がコンピュータコードであることを示す
col 表の縦列を表す
colgroup 表の縦列をグループ化する
data 要素内容を機械が読み取り可能な形式で指定する際に使用
datalist 入力欄に入力候補を一覧として表示
dd 定義リストにおける用語の説明文を表す
del 後から削除された文字列であることを示す
details ユーザーが操作することで詳細情報が表示されるウィジェットを作成
dfn 文章の中で定義されている用語部分を示す
dialog ダイアログボックスやウィンドウなど、ユーザーが操作可能な対話型の部品を表す
div 特定の意味を持たず、ブロックレベル要素として指定するために汎用的に使用
dl 特用語とその説明をセットにした形式の定義リストを表す
dt 定義リスト内の用語部分を示す
em 文字列を強調する
embed 文書の中に外部コンテンツを埋め込む
fields フォームの内容をグループ化する
figcaption 親のfigure要素に説明文を追加する
figure 文書の本題から参照される図表であることを示す
footer ページやセクションのフッター部分をグループ化する
form ユーザーが入力した情報を送信することができる入力フオームを示す
h1...h6 セクションの見出しを最もレベルの高いh１から順番に６段階で表す
head 文書自身に関する情報を定義する
header ページやセクションのヘッダー部分をグループ化する
hr 段落レベルでの内容の区切りを示す
html HTML文書における全ての要素を含む、最上位の要素
