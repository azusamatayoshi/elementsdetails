HTML elements

Hello, world!

HTML文書を構成する基本要素
html HTML文書における全ての要素を含む、最上位の要素
head 文書自身に関する情報を定義する
body ブラウザに表示される文書内容を表す
文書情報
meta 様々な種類のメタデータ（その文書の情報）を表す
title 検索結果やブックマーク、ページのタブなどに表示される文書のタイトルを表す
base 文書中で使用される全ての相対URLの基準となるURLを指定する
link 文書と外部ファイルとの関係性を表す
style 文書にスタイルを適用する
script JavaScriptなどのスクリプトを埋め込む
noscript スクリプトが無効の場合に表示される代替内容を表す
セクション
main 文書内の主要な内容を表す
section 文書の中での章や節など、ひとつのまとまりを表す
article それだけで完結している内容の記事を表す
aside 文書のメインコンテンツとは関連性の低いセクションであることを表す
nav 主要なナビゲーションリンクを表す
header ページやセクションのヘッダー部分をグループ化する
footer ページやセクションのフッター部分をグループ化する
address 連絡先を表す
h1...h6 セクションの見出しを最もレベルの高いh１から順番に６段階で表す
hgroup ・・・
コンテンツをグループ分けする要素
div 特定の意味を持たず、ブロックレベル要素として指定するために汎用的に使用
p 段落を表す
blockquote その段落が引用されたものであることを表す
pre 空白や改行などがそのまま表示される、整形済みテキストを表す
figure 文書の本題から参照される図表であることを表す
figcaption 親のfigure要素（図表）に説明文を追加する
hr 段落レベルでの内容の区切りを表す
文字列に意味づけをする要素
span 特定の意味を持たず、インライン要素として指定するために汎用的に使用
a ハイパーリンクを設定する
strong 重要性の高い文字列を示す
em 文字列を強調する
mark 目立たせるためにハイライトされた文字列を示す
small 免責事項や著作権表記など、付帯情報を小さいフォントで表す
b その文字列にに読者の注目を集める意味で使用し、特に重要性を示すものではない
i 他の部分とは意味合いが違っている文字列を示す
time 日付や時刻を示す
q 短めの引用箇所を示す
cite 文書の中で引用または言及した作品（blockqupote要素・q要素）のタイトルを示す
dfn 文章の中で定義されている用語部分を示す
abbr 略語であることを示す
sup 上付き文字を示す
sub 下付き文字を示す
ruby ルビを振る範囲を指定する
rt ルビとして表示される内容を示す
rp ルビ表示に未対応のブラウザにおいて括弧内にルビを表示する
bdi 要素内容の文字列が他とは別の文字表記の方向であることを示す
bdo 現在の文字表記の方向を上書きし、要素内容の文字列の表記方向を明示的に指定する
br 改行を示す
wbr 改行しても良い位置を示す際に使用
code 要素内容の文字列がコンピュータコードであることを示す
var ・・・
samp コンピュータプログラムによって出力される文字列の例を示す
kbd ユーザーが入力する文字列を示す
data 要素内容を機械が読み取り可能な形式で指定する際に使用
s 正確ではなくなった、無効な文字列を示す
u ・・・
del 後から削除された文字列であることを示す
ins 後から追加された文字列を示す
リスト ・・・
ul 一般的な箇条書きなど、項目の順番に特に意味がないリストを表す
ol 項目の順番に意味のあるリストを表す
li リスト（ol要素・ul要素）内の項目を表す
dl 用語（dt要素）とその説明（dd要素）をセットにした形式の定義リストを表す
dt 定義リスト（dl要素）内の用語部分を表す
dd 定義リスト（dl要素）における用語（dt要素）の説明文を表す
コンテンツの埋め込み
img 文書の中に画像を埋め込む
audio 文書の中に音声データを埋め込む
video 文書の中に動画を再生するプレイヤーを設置する
picture レスポンシブデザインなど、ブラウザの条件によって複数の画像を用意する際に使用
source 埋め込む音声や動画データを複数指定する
track 音声や動画データに、字幕などのテキストトラックを埋め込む
canvas 図形やアニメーションを表示させるために使用
object 文書の中に外部ファイルのコンテンツを埋め込む
param object要素（外部ファイルのコンテンツ）の引数を定義する
embed 文書の中に外部コンテンツを埋め込む
iframe 文書の中に別のHTML文書を埋め込むインラインフレームを表示
map イメージマップを表す
area イメージマップ（map要素）における領域やリンク先などを設定
表
table 表を作成する際に使用
caption 表（table要素）のタイトルを示す
tr 表の横１行全体を表す
th 表の見出しセルを表す
td データを入れるための普通のセルを表す
thead 表のヘッダー部分の行（tr要素）をグループ化する
tbody 表の本体部分の行（tr要素）をグループ化する
tfoot 表のフッター部分の行（tr要素）をグループ化する
col 表の縦列を表す
colgroup 表の縦列（col要素）をグループ化する
フォーム
form ユーザーが入力した情報を送信することができる入力フオームを表す
input さまざまな入力欄や選択部品などを表す
textarea 複数行の入力欄を表す
button クリックできるボタンを表す
select 複数の項目から選択するプルダウンメニューやリストボックスを表示
option select要素・datalist要素内での選択肢を示す
optgroup select要素内の選択肢（option要素）をグループ化する
datalist 入力欄に入力候補を一覧（option要素）として表示
fieldset フォームの内容をグループ化する
legend グループ化したフォーム（fieldset要素）のキャプションを表す
label フォーム部品の項目名を表す
output 計算結果を表示する
progress プログレスバーを表示し、タスクの進捗状況を表す
meter 特定の範囲において分量がどれだけかを表す
インタラクティブ
details ユーザーが操作することで詳細情報が表示されるウィジェットを作成
summary 開閉式ウィジェット（details要素）における詳細情報の見出し、説明を表す
dialog ダイアログボックスやウィンドウなど、ユーザーが操作可能な対話型の部品を表す
